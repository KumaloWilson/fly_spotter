## ğŸš€ Quick Start Guide

### Prerequisites
- Install Flutter SDK
- Set up Firebase project
- Configure Google Maps API key

### Installation Steps
1. Clone the repository
   ```bash
   git clone https://github.com/KumaloWilson/fly_spotter.git
   ```

   ```bash
   flutter run
   ```
